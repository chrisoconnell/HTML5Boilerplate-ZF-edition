HTML5Boilerplate-ZF-edition by chrisoconnell

Bringing the awesome-ness of HTML5 Boilerplate and the Zend Framework together!

Dependencies

Zend Framework version 1.8 or higher.

Install

Find the location of your Zend Framework install. Mine is at "/usr/local/zend/share/ZendFramework" but yours could be different. Now go to "library/Zend/Tool/" so your final path should be "/path/to/ZendFramework/library/Zend/Tool/" Make a backup of the Project directory (just in case). Replace the "Project.php" file in the "Project/Provider" directory with the one from HTML5Boilerplate-ZF-edition. Copy every file beginning with "H5bp" into "Project/Context/Zf" So if you are doing everything from the command line, it will look like: cd /path/to/ZendFramework/library/Zend/Tool/ cp -r Project Project-old cp /path/to/HTML5Boilerplate-ZF-edition/Project.php Project/Provider/ cp /path/to/HTML5Boilerplate-ZF-edition/H5bp* Project/Context/Zf/
